# Реализованные задачи

## Метаданные и структура
- Подсистемы: `НСИ`, `Закупки`, `Продажи`, `Склад`, `Отчеты`, `Администрирование`, `Интеграция`.
- Справочники: `Номенклатура`, `Склады`, `Контрагенты`, `Сотрудники`, `ВидыЦен`.
- Перечисления: `СтатусыЗаказов`, `ВидыОперацийСклада`.
- Документы: `ПоступлениеТоваров`, `ЗаказКлиента`, `РеализацияТоваров`, `ПеремещениеТоваров`, `СписаниеТоваров`.
- Регистры: `ОстаткиТоваров`, `РезервыТоваров`, `ЦеныНоменклатуры`.

## Бизнес-логика
- Приход товара увеличивает остатки.
- Реализация и списание уменьшают остатки.
- Заказ клиента резервирует товар.
- Реализация снимает резерв по заказу-основанию.
- Перемещение переносит остатки между складами.
- Встроены проверки на недостаточный остаток и недостаточный резерв.

## Отчетность и интеграция
- Реализованы отчеты на СКД:
  - `ОстаткиТоваровПоСкладам`
  - `ПродажиЗаПериод`
  - `ТоварыВРезерве`
  - `ТопТоваров`
- Реализована обработка `ЗагрузкаЗаказовИзJSON`.
- Добавлены печатные формы для заказа и реализации.

## Роли и доступ
- `Менеджер`
- `Кладовщик`
- `Администратор`
