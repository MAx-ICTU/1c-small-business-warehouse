# Общее описание проекта

## Цель
Мини-конфигурация 1С для малого бизнеса с фокусом на учет заказов и складских операций:
- поступление и отгрузка товаров;
- резервирование под заказ клиента;
- контроль остатков и резервов;
- управленческая отчетность;
- базовая интеграция загрузки заказов из JSON.

## Предметная область
- НСИ: номенклатура, склады, контрагенты, сотрудники, виды цен.
- Продажи: заказ клиента, реализация товаров.
- Закупки: поступление товаров.
- Склад: перемещение и списание.
- Отчеты: остатки, продажи, резервы, топ товаров.

## Что делает проект полезным
- Покрывает реальный сквозной процесс от прихода товара до реализации.
- Показывает прикладную бизнес-логику проведения документов.
- Демонстрирует работу с регистрами, отчетами СКД, печатными формами и интеграцией JSON.
- Может использоваться как стартовый шаблон для доработки под конкретный бизнес.
