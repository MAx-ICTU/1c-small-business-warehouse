# Как использовать проект

## 1. Назначение
Проект предназначен как готовый пример мини-конфигурации 1С для автоматизации заказов и складского учета в малом бизнесе.

## 2. Базовый сценарий работы
1. Заполнить НСИ: номенклатуру, склады, контрагентов, виды цен.
2. Оформить поступление товаров на склад.
3. Создать заказ клиента и зарезервировать товар.
4. Создать реализацию по заказу.
5. При необходимости выполнить перемещение или списание.
6. Сформировать отчеты СКД для анализа.

## 3. Проверка интеграции JSON
1. Открыть обработку `ЗагрузкаЗаказовИзJSON`.
2. Указать файл примера: `samples/orders/order-001.json`.
3. Выполнить загрузку и открыть созданный документ `ЗаказКлиента`.

## 4. Что смотреть в первую очередь
- Контроли остатков и резервов при проведении документов.
- Регистр цен и подстановка цены в документы продаж.
- Отчеты СКД по остаткам, продажам и резервам.
- Печатные формы заказа и реализации.
