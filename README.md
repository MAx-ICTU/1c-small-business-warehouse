## Назначение проекта
Конфигурация покрывает базовые процессы закупок, продаж и склада:
- поступление товара;
- резервирование под заказ клиента;
- реализация, перемещение и списание;
- контроль остатков и резервов;
- отчеты на СКД;
- загрузка заказов из JSON.
